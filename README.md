# Jai AI – Intelligent Tutoring System for Remote India

### Problem Statement

In many rural and remote areas, students face:

* Lack of qualified teachers
* Limited access to quality study materials
* Poor internet connectivity
* High cost of advanced AI tools

Modern AI tutors require high internet speed and expensive resources, making them inaccessible for many students.

### Solution – Jai AI

Jai AI is a lightweight intelligent tutoring system designed to:

* Provide accurate and simple explanations
* Work with low internet and low computing power
* Support state-board syllabus learning
* Deliver personalized learning experiences

### Objectives
* Make education accessible to rural students
* Reduce dependency on high-cost AI systems
* Provide concept-based learning instead of memorization
* Enable learning even with limited connectivity

### Key Features

#### AI-Based Question Answering
* Students can ask questions in natural language
* System provides clear, step-by-step answers
* Focus on simple explanations

#### Curriculum-Aligned Learning
* Answers follow school syllabus (state board / NCERT)
* Avoids unnecessary advanced content
* Helps in exam preparation

#### Low Resource Optimization
* Uses efficient models or optimized APIs
* Reduces data usage and processing time
* Works on low-end devices

#### Personalized Learning
* Adapts answers based on student level
* Provides simplified or detailed explanations

#### Simple User Interface
* Easy-to-use design
* Works on mobile, laptop, and basic systems

### Modules Description

#### Frontend Module
* Handles user interaction
* Displays questions and answers
* Technologies: HTML, CSS, JavaScript / React

#### Backend Module
* Processes user requests
* Connects frontend with AI module
* Handles API requests and data processing
* Technologies: Node.js / Python

#### AI Processing Module
* Core logic of the system
* Performs NLP and answer generation
* Uses local models or APIs

#### Data Module
* Stores textbooks, notes, and questions
* Provides academic knowledge base

#### Utility Module
* Handles text processing
* Input validation
* Response formatting

### Workflow

1. Student opens Jai AI
2. Enters a question
3. Frontend sends request to backend
4. Backend processes input
5. AI module generates answer
6. Response is sent back to frontend
7. Student views the answer

### Installation Guide
#### Step 1: Clone Repository

git clone https://github.com/jenohsam2006/JaiAI.git

cd JaiAI

#### Step 2: Install Dependencies

* Node.js:

npm install

* Python:

pip install -r requirements.txt

#### Step 3: Run Application

* Node.js:

npm start

* Python:

python app.py
